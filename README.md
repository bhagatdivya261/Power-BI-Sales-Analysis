# Power BI Sales & User Analysis Project

## Project Description
This project is created using Power BI to analyze monthly revenue, users, devices, countries, gender, and subscription types.

## Tools Used
- Power BI Desktop
- MS Excel (Dataset)
- GitHub (Project Hosting)

## Key Analysis
- Monthly revenue by device and subscription
- Gender-wise revenue and user count
- Country-wise user distribution
- Subscription type comparison (Basic, Standard, Premium)
- Device usage analysis

## Files Included
- Power BI Dashboard (.pbix)
- Project Report (PDF)
- Dashboard Screenshots

## Conclusion
This dashboard helps understand customer behavior, revenue trends, and device usage patterns, which supports business decision-making.
